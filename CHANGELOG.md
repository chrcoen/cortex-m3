# Changelog
All notable changes to this project will be documented in this file.

## [v1.0.0]
### Added
* Initial platform
  * ARM Cortex M3 QBox (Containing ARM Cortex M3 + System controller) TLM2C and SimpleCPU
  * SimpleMemory
  * AHB GPIO
  * APB UART
  * APB Dual-Input Timers
  * APB Timer
  * APB Watchdog
  * PL022

* Backend
  * TCPSerial
